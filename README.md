# AI Chat Assistant

A full-stack AI chatbot built with Node.js, Express, Gemini API and SQLite.

## Features
- AI chat powered by Gemini
- Markdown rendering
- Syntax-highlighted code blocks
- Copy code button
- Typing animation
- Stop generation button
- Chat history UI

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express  
AI: Gemini API  
Database: SQLite
